How to use this Todo List

  • Type your task in the text box and click "Add Task" or press Enter
  • Each task will appear in the list with a delete button
  • Click the delete button to remove a specific task
  • Use "Clear All" to remove all tasks at once
  • Your tasks are stored in the FastAPI backend server
  • Make sure the backend server is running at http://localhost:8000
